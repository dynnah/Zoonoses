### Instalação
<br />

- Fazer um git clone do repositório
- Entrar na pasta do projeto (via terminal) e executar o seguinte comando:
  - yarn install
- Após finalização do comando, executar o seguinte comando para ver o projeto rodando:
  - yarn start
